HUB75 Audio Matrix

A live audio visualizer and 64x64 HUB75 RGB matrix simulator with GPIO trace decoding, ScreenCaptureKit audio input, and Metal rendering.

Read project notes View pin map

Simulator

Built for signal-level visibility

64x64 framebuffer

Models a logical RGB matrix with a 1/32-scan HUB75 refresh path.

Audio analysis

Tracks RMS, peak, envelope, FFT spectrum, frequency bands, smoothing, noise floor, and AGC.

GPIO trace decoding

Reads CLK, LAT, OE, row addressing, and upper/lower RGB data activity.
